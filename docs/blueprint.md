# **App Name**: ClientsFinding

## Core Features:

- Local Business Search: Allows users to input business types (e.g., 'plumber', 'doctor') and a specific location to search for local business leads.
- Interactive Map Display: Visualizes search results as markers on an interactive Google-style map, similar to app.maplent.com.
- Business Detail View: Displays detailed information for selected businesses, including address, contact, and potentially user reviews or ratings.
- Lead Management Lists: Enables users to save identified leads into custom-named lists for organization and future reference.
- Smart Search Suggestion Tool: Leverages AI to suggest relevant search terms or categories as the user types, enhancing discovery.
- Lead Export Functionality: Allows users to export their saved lead lists in common data formats (e.g., CSV, JSON) for external use.

## Style Guidelines:

- Primary action color: A professional and trustworthy medium-saturated blue (#247ECC) for interactive elements and key highlights.
- Background color: A very light, almost white, desaturated blue (#F4F8FA) providing a clean and spacious canvas for content, derived from the primary hue.
- Accent color: A vibrant turquoise (#4DDBDB) to draw attention to secondary actions, notifications, or important data points, creating a strong contrast with the primary blue.
- Body and headline font: 'Inter' (sans-serif) for a modern, objective, and highly readable experience across all text, reflecting a data-driven tool.
- Use clear, concise line icons that convey functionality without clutter, consistent with the minimalist yet professional aesthetic shown in the example, such as the magnifying glass.
- Employ a distinct two-column layout with a dark, persistent left sidebar for navigation and account management, contrasting with a spacious, light main content area for search and map interactions.
- Incorporate subtle, smooth animations for map panning, zooming, and when displaying or filtering search results, to provide a responsive and polished user experience.
